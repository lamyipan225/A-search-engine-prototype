files:
lib/jdbm-1.0.jar	//java library for indexing
lib/htmlparser.jar	//java library for extracting info. from web page
stopwords.txt  		//stopword list for stopword removal
IRUtilities/Crawler.java		//A java program craws information from webpage and performs stopword removal. Indexed result is saved in Spider_result.txt.
IRUtilities/Document.java		//A java program helps compare documents¡¦ score.
IRUtilities/InvertedIndex.java	//Perform title search and content search. Evaluate document and construct their ranking.
IRUtilities/Porter.java		//Porter Algorithms implemented by Fotis Lazarini for stemming.
IRUtilities/SearchEngine.java	//Receive queries and implement searching in inverted list after received queries.
ti.db			//A db file for title indexing.
Stopwords.txt		//A hardcoded string list for stopwords removel for both indexing and query preprocesing.
spider_result.txt	//A text file records the indexing reult.
query_result.txt	//A text file records the query result which generated by SearchEngine.class. 
number of documents.txt	//A text file recorded number of webpage in our system.
index.html		//A html file(starting page of our search engine) for user to input query in webbrower.
search.jsp		//A jsp file for system to call SearchEngine.class, then present the result.

----------------------------------------------------------------------------------------------------------------------------------------------------
Compile and Index:
1.compile all *.java to generate classes

2.Run Crawler.java for indexing
>>javac -cp .:lib/jdbm-1.0:lib/htmlparser.jar:. IRUtilities/Crawler.java
>>java -cp .:lib/jdbm-1.0:lib/htmlparser.jar:. IRUtilities/Crawler

3.index result
all result is in spider_result.txt

----------------------------------------------------------------------------------------------------------------------------------------------------
Searching:
Run SearchEngine.java for searching
>>javac -cp .:lib/jdbm-1.0:lib/htmlparser.jar:. IRUtilities/SearchEngine.java
>>java -cp .:lib/jdbm-1.0:lib/htmlparser.jar:. IRUtilities/SearchEngine.java "query"

----------------------------------------------------------------------------------------------------------------------------------------------------
Installation Procedure
Users only need to go into the project directory(FinalPhase/) and input keywords to search.
Result webpages with titles will be appeared in your Web browser.
You may also check the result in query_result.txt.
